## Installation and Setup

1. First, clone the project:

```bash
git clone https://github.com/Alisamavat-dev/wavnix-React
cd react
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

4. To build the production version:

```bash
npm run build
```
